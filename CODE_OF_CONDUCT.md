# Code of Conduct

## Our Standards

This project exists to help people improve their content. We expect all participants to:

- Be respectful and constructive in issues and pull requests
- Focus feedback on the skill's output and methodology, not on individuals
- Accept that content scoring is opinionated by design — debate the criteria, not the person

## Unacceptable Behavior

- Harassment, insults, or personal attacks
- Spam or self-promotion unrelated to the project
- Publishing others' private information

## Enforcement

Violations can be reported to hello@whystrohm.com. Maintainers will review and take appropriate action, including removing comments or banning repeat offenders.

## Scope

This code of conduct applies to all project spaces: issues, pull requests, and discussions.
